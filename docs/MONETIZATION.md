# Идеи монетизации

> Варианты монетизации экосистемы SkandiKlubb. Приоритет — после создания ценности для жителей.

---

## 1. Реклама на сайте

**Модель:** размещение рекламы локальных сервисов.

**Откуда брать рекламодателей:**
- Парсинг чатов → кто рекламирует услуги → предложить размещение на сайте
- Локальные бизнесы Петроградского района

**Целевая аудитория:**
- 1222 квартиры, Петроградский район
- Платежеспособные жители (ЖК комфорт-класса, Bonava)

**Форматы:**
- Баннер на сайте
- Карточка в разделе "Полезное"
- Спецпредложение для жителей ЖК

**Ценообразование:** определить после набора трафика (Фаза 2-3).

---

## 2. Доска мастеров

**Модель:** маркетплейс проверенных специалистов.

**Категории:**
- Сантехники
- Электрики
- Клининг
- Ремонт квартир
- Установка/обслуживание рекуператоров
- Кондиционеры
- Мебель на заказ

**Как работает:**
1. Жители рекомендуют мастеров в чатах (парсинг)
2. Мастер размещается на сайте с рейтингом и отзывами
3. Жители находят мастера через сайт/бота

**Монетизация:**
- Бесплатное размещение (базовое)
- Платное выделение / верхняя позиция
- Комиссия с заказов (сложнее, на перспективу)

---

## 3. Генератор жалоб (freemium)

**Бесплатно:**
- Базовые шаблоны жалоб (текст для копирования)
- Информация о проблемах и инстанциях

**Платно (подписка или разово):**
- PDF с оформлением (шапка, подпись, реквизиты)
- Автоматическая отправка по email в инстанции
- Пакет "по всем инстанциям" (одна проблема → 5+ документов)
- Трекинг: отслеживание статуса обращения
- Юридическая консультация (партнёрство с юристами)

**Ценовые гипотезы:**
- Один документ PDF: 100-200 руб.
- Пакет "все инстанции": 500-1000 руб.
- Подписка "безлимит": 300-500 руб./мес.

---

## 4. Сезонные услуги (групповые заказы)

**Модель:** организация коллективных заказов со скидкой.

**Услуги:**
- Мойка фасадов / окон (скалолазы, весна)
- Чистка вентиляции
- Обслуживание рекуператоров
- Промывка системы отопления
- Дезинсекция / дератизация
- Чистка ливнёвки

**Как работает:**
1. Сбор заявок от жителей (через сайт/бота)
2. Формирование группы (минимум N квартир)
3. Договорённость с подрядчиком (скидка за объём)
4. Организация, контроль качества

**Монетизация:**
- Комиссия с подрядчика (10-20%)
- Организационный сбор с жителей (фиксированный)

---

## 5. Масштабирование

### Вариант A: Соседние ЖК
- Петроградский район → другие ЖК с проблемными УК
- Та же ЭКВИДА-СЕРВИС управляет другими домами?
- Адаптация сайта под другой дом (минимальные изменения)

### Вариант B: White-label
- "Экосистема для ЖК" как продукт
- Инициативная группа другого дома получает готовый инструмент
- Настройка: адрес, УК, проблемы, инстанции

### Вариант C: SaaS для УК/ТСЖ
- Подписка для управляющих компаний или ТСЖ
- Инструмент коммуникации с жителями
- Управление обращениями, голосованиями

## Важно

- Монетизация — НЕ приоритет на Фазах 0-2
- Сначала создать ценность для жителей, потом думать о деньгах
- Генератор жалоб — единственная фича, где монетизация встроена в продукт
- Всё остальное — после набора аудитории
