## Порядок обращений (эскалация)

```
1. УК (ЭКВИДА-СЕРВИС) — письменное обращение
   ↓ нет ответа 10 дней / отказ
2. ГЖИ + ГИС ЖКХ — параллельно
   ↓ нет результата 30 дней
3. Прокуратура + Роспотребнадзор (если применимо)
   ↓ нет результата
4. Суд — исковое заявление
```
