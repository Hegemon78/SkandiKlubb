## Как работать с этим файлом

1. **Начало сессии:** прочитай ROADMAP.md, найди свои задачи
2. **Взятие задачи:** поменяй статус ⬜ → 🔄, укажи сессию
3. **Завершение:** поменяй статус 🔄 → ✅
4. **Блокировка:** поменяй статус → ⏸️, опиши причину
5. **Конфликт:** если две сессии редактируют один файл — ROADMAP.md решает приоритет
