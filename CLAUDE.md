# CLAUDE.md — SkandiKlubb

> Экосистема для жителей ЖК Skandi Klubb: сайт + Telegram-бот + генератор жалоб + парсинг чатов.

---

## Назначение проекта

Полноценная экосистема для жителей ЖК Skandi Klubb (Санкт-Петербург, Аптекарский пр-кт, 18):
- **Сайт** — информация о доме, проблемах, контактах, правах жителей
- **Генератор жалоб** — AI-генерация документов для 6 инстанций (встроен в сайт)
- **Telegram-бот** — альтернативный интерфейс для генерации жалоб (Фаза 3)
- **Парсинг Telegram-чатов** — сбор и анализ данных от жителей (30,591 msg)

## Координация параллельных сессий

**ROADMAP.md** — главный файл для координации. Каждая сессия Claude:
1. Читает ROADMAP.md → берёт свою задачу
2. Читает соответствующий `docs/*.md` для контекста
3. Работает автономно
4. Отмечает прогресс в ROADMAP.md

**Правила:**
- Не трогать файлы другой сессии без необходимости
- При конфликтах — ROADMAP.md имеет приоритет
- Изменения в docs/ — только по согласованию

## Правила для сессий

1. **Сначала читай ROADMAP.md** — определи что нужно делать
2. **Читай docs/ для контекста** — не начинай без понимания плана
3. **Данные берём из УК ОСС** — `~/УК ОСС/` как source of truth
4. **Отмечай прогресс** — обновляй ROADMAP.md при завершении задач
5. **Не коммить secrets** — .env, *.session, data/ в .gitignore
6. **Юридическая точность** — ссылки на законы должны быть корректны
7. **Палитра** — только sage green / linen / amber, zero blue-* classes
